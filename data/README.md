# Data Folder

Place input spreadsheets and completed grader files here.

## Suggested contents

- `Anaes_Questions.xlsx`
- `Pharma_Questions.xlsx`
- `Ophthalmology_Questions.xlsx`
- `score_grades_long_format.csv`

## Question file format

Each file should contain one question column, for example:

- `Question`
- `Questions`

## Post-grading format

Recommended long-format schema:

- `domain`
- `model`
- `response_id`
- `Safety`
- `Consensus`
- `Objectivity`
- `Reproducibility`
- `Explainability`
