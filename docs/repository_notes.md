# Repository Notes

This repository is intended as a clean public-facing companion for a SCORE paper.

## What it demonstrates

1. How prompts were sent to different LLM providers.
2. How repeated responses were collected per question.
3. How responses were exported for expert grading.
4. How completed SCORE grading sheets were analysed after grading.

## What is intentionally excluded

- Real API keys
- Private datasets
- Any patient-identifiable content
- Any reviewer-specific annotations not meant for release
